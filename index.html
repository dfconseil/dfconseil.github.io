Qui sommes-nous ?

DF Conseil vous accompagne dans vos projets informatiques, de la conception jusqu'à la maintenance au long cours. Nous intervenons également sur des problématiques ponctuelles de sécurtité, de performance, jusqu'à la refonte complète d'une application full-stack.

Nos technologies et domaines de compétence

Nous maîtrisons parfaitement les technologies sur lesquelles nous intervenons, et refusons les missions sur lesquelles nous ne serions pas en mesure de garantir un résultat irréprochable.

Nous intervenons essentiellement sur les technologies suivantes :

  • Ruby et Ruby on Rails
  • Bases de données relationnelles MySQL, PostgreSQL, Oracle, Microsoft SQL Server
  • Environnements Unix, en serveurs dédiés ou en environnement cloud
